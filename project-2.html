Metapmate

This page contains the case study of Metapmate which includes the Project Overview, App Snapshots, and Tools Used.

Project Overview

Metapmate is a dating application and parsing system that I created for a client in 2021. Metapmate works similar to other dating applications. Nevertheless, it introduces metaphysical concept of human to human relationships. Without a team, I took more than a year to complete the project, especially during my studies.

Metapmate is a hybrid-platform application meaning that it can run in both iOS and Android environment. The application offers a vast number of features, spanning from authentication, to in-app subscriptions, and notifications service. I implemented the MVVM system architecture along with Dependency Injection design pattern. During the development phase, I referred to the application User Interface prototype provided by my client using Adobe XD.
