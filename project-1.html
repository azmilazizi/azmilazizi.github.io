ROOTMYBOX Report System

This page contains the case study of RMB Reporting System which includes the Project Overview, System Snapshots, and Tools Used.

Project Overview

RMB Reporting System is a reporting manaagement and parsing system that I created where the company can manage, import pentesting vulnerability and findings into the system and eventually parse it into a report.

The system is developed during my internship at ROOTMYBOX. The system is an automated solution for the time consuming process of post testing reporting. RMB Report System provides automated imports from CSV files, manageability, and report parsing. I took User Interface ideas by visiting different types of websites of the same industry and even visited platforms like dribbble to get inspiration for building a good User Interface.
